# Context Recovery & Handoff Protocol

> Agents don’t degrade. Their memory does.

Agents feel like they “get worse” over time.

They don’t.

Context silently accumulates:
- slower responses  
- worse reasoning  
- drift  

Most people think it’s throttling.

It’s not.

It’s memory bloat.

---

## Solution

Detect → Handoff → Compact → Restore
